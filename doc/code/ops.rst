pennylane_ionq.ops
==================

IonQ's native trapped-ion gates as PennyLane operations. Phases (``phi``,
``phi0``, ``phi1``) are in **turns**; interaction parameters (``theta``,
``angle``) are in **units of** :math:`\pi`.

These gates are accepted directly when the device is constructed with
``gateset="native"``.
