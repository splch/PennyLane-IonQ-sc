pennylane_ionq
==============

The top-level module re-exports the device classes (``IonQDevice``,
``IonQSimulatorDevice``, ``IonQQPUDevice``) and the four IonQ-native gate
operations. Most users access the devices through ``qml.device(...)``;
see :doc:`/devices`.
